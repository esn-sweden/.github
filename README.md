# ESN Sweden

<img src="https://accounts.esn.org/modules/custom/esn_accounts_groups/images/logos/SE/SE.png" alt="ESN Sweden logo" height="200px">

Welcome to ESN Sweden's Github page!

We are a non-profit part of the Erasmus Student Network working for the integration and wellbeing of international students.

## Contribute

We welcome any suggestions for improvements on our public repos. Don't hesitate to open an issue!

## Contact

You can contact the current IT Manager by sending an email to wpa@esnsweden.org or the board at board@esnsweden.org. You can find up-to-date contact information for the rest of our members on [our website](https://esnsweden.org/team/).
